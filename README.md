## Installation

Available via JCenter.

### Gradle
```
implementation 'de.nerden:kafka-streams-contrib:0.3.5'
```

### Maven
```
<dependency>
  <groupId>de.nerden</groupId>
  <artifactId>kafka-streams-contrib</artifactId>
  <version>0.3.5</version>
  <type>pom</type>
</dependency>
```
